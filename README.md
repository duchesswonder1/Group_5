## Academics
This section of the website was created by Alinda Sasha.

### Description
The Academics page is a recreation of the St. Mary's College Kisubi (SMACK) academic section. It was built using only HTML and CSS, maintaining the visual style and layout of the original website, The Acdemics page serves as a digital representation of the school's educational pathways , providing students and visitors with information about the dual_level O-level competency curriculum and the rigorous A-level pre-university combinations. It highlights the importance of the instituition  as the center for academic excellence, showcasing its strong track record in science education alongside its state of-the-art Science Block facilities. The page was carefully designed to reflect the professional and academic atmosphere of SMACK, using a clean and organised layout that makes it easy for users to navigate and find information they need.
